# 📘 Assignment: Data Algorithms with pandas

## 🎯 Objective

Use the pandas library to practice loading tabular data, filtering and sorting records, and computing aggregated summaries. This assignment helps students apply data algorithms to real datasets.

## 📝 Tasks

### 🛠️ Load and Explore Data

#### Description
Read the dataset from a CSV file and inspect its contents to understand the available fields.

#### Requirements
Completed program should:

- Load `data.csv` with pandas.
- Display the first few rows using `head()`.
- Print the column names and row count.

### 🛠️ Filter and Sort Records

#### Description
Filter the dataset to find matching rows and sort the results by a useful column.

#### Requirements
Completed program should:

- Filter records by a condition, such as category or price range.
- Sort the filtered records by a numeric column like `price` or `quantity`.
- Print the filtered and sorted results clearly.

### 🛠️ Aggregate and Summarize

#### Description
Compute summary statistics and group data to find insights.

#### Requirements
Completed program should:

- Group records by a category column.
- Calculate totals or averages for each group.
- Print a summary table that shows aggregated results.
